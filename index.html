GabayRa

Your Medication Companion — React Native app + IoT pillbox + risk model

A medication adherence system built for families managing multiple patients at once. The React Native app pairs with an IoT pillbox over MQTT to log doses in real time, tracks per-patient compliance, and uses an XGBoost model to flag missed-dose risk before it becomes a pattern.
